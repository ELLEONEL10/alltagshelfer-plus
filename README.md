# Alltagshelfer Plus (React + Vite)

Professional website for first-aid training and event medical coverage.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- i18next (DE/EN)
- Firebase (optional modules already included)

## Run Locally

1. Install dependencies:
   npm install
2. Start development server:
   npm run dev

## Production Build

Build for publish/deploy:

npm run build

Preview production build locally:

npm run preview

## Languages

- German (default)
- English

## Deploy

The app outputs static files to dist after running npm run build and can be deployed to:

- Firebase Hosting
- Netlify
- Vercel
- Any static host
